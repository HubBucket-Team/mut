===========
mut-publish
===========

Synopsis
--------

``mut-publish`` is a publication tool that synchronizes a filesystem directory
with an Amazon S3 bucket.

Usage
-----

.. code-block:: sh

   Usage: {name} <source> <bucket> --prefix=prefix (--stage|--deploy)
                    [--redirects=htaccess]
                    [--redirect-prefixes=prefixes]...
                    [--destage] [--dry-run] [--verbose]
