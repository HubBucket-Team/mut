YAML Format
-----------

.. code-block:: yaml

   title: <string>
   stepnum: <integer>
   content: <string>
   post: <string>
   pre: <string>
   level: <integer>

   action:
     code: <string>
     copyable: <boolean>
     content: <string>
     heading: <string>
     language: <string>
     post: <string>
     pre: <string>

   replacement:
     <old>: <new>

   inherit:
     file: <path>
     ref: <string>

``action`` may also be either a string or a list. If it is a string, this is
shorthand for filling out the ``content`` field. If it is a list, then a
sequence of actions are made.
